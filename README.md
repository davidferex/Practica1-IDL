# Practica 1 Infraestructuras para Deep Learning

Autores: David Fernández Expósito, Ángel Martínez Sánchez, Javier Polo Gambín


En este repositorio mostramos los experimentos realizados para la primera práctica de la asignatura Infraestructuras para Deep Learning, del máster en Inteligencia Artificial de la Universidad de Murcia.

El repositorio se divide en dos directorios:
- Distilbert: Aquí se incluyen los experimentos realizados con el modelo Distilbert y accelerate durante su proceso de entrenamiento.
- Whisper-Tiny: Aquí se incluyen los experimentos realizados con el modelo Whisper-Tiny de OpenAI y accelerate durante su proceso de inferencia.
